## Synopsis

Enter a URL to scan, a depth and the app will crawl all the links it can find based on the depth  

## Installation

Install the [requirements](#requirements)
```bash
pip install PySide6
pip install BeautifulSoup
pip install qdarkstyle
pip install requests
```
